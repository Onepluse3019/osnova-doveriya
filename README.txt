Сайт «Основа доверия»

Как открыть:
1. Распакуйте архив.
2. Откройте файл index.html в браузере.

Структура:
- index.html — содержимое сайта
- style.css — оформление и адаптивность
- script.js — мобильное меню и анимации
- assets/images — изображения
